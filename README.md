# FengQiReader

仿照"任阅"的追书App。

详见原作者项目地址:[任阅](https://github.com/JustWayward/BookReader)

风骑小说是一款拥有海量小说与连接在线站点的手机阅读软件平台。
百万本热门电视剧小说，经典名著尽情畅读，内容覆盖总裁豪门小说、言情小说、穿越架空小说、玄幻小说、青春校园小说、修仙小说、悬疑小说、同人小说、名著、各种类型。你想看的小说，这都有！
风骑小说是一款基于互联网的小说搜索下载工具，同时优质的资源及阅读体验，为您提供最精致的阅读服务！
特色功能：
1. 内置多个小说站点网站，总有一个是您需要的。
2. 支持各小说网站海量小说搜索及阅读，完全免费。
3. 支持下载部分章节,为您节省流量。
4.支持本地书籍、网络书籍。
5.流畅的下载管理，能暂停，继续，删除各种下载任务。
6.智能导入文件设计,功能齐全,操作简单,界面美观大方!
7.各种主题,字体,亮度,颜色,方向、阅读进度等随意调整。
